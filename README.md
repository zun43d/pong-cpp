# Pong (2019)

A simple C++ clone of Pong.

## Required libaries
- SDL2
- SDL2_image
- SDL2_ttf
- SDL2_mixer

## To run with g++
```
g++ ./Main.cpp -lSDL2 -lSDL2_ttf -lSDL2_mixer -lSDL2_image
```
